Config

I've been too lazy to implement config... it'll come to you soon(TM).

For now, you can open up the JavaScript console in your browser and eddit the config object.

The most interesting config property is interval (config.interval, default is 60000). That represents the interval between background checks in milliseconds. Be sure to start and stop the background with the functions `status.background(false)` and `status.background(true)`.
